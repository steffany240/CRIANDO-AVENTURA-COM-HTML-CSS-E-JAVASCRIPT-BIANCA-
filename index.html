O jogador encontra um mapa antigo em uma biblioteca esquecida. O mapa possui símbolos enigmáticos e coordenadas que precisam ser decifradas. Desafio: O jogador deve decifrar um código com base nas pistas do mapa. O código é uma sequência de letras que formam uma palavra-chave. Exemplo de Enigma: "A palavra que você procura está escondida nas primeiras letras dos pontos cardeais."

O jogador encontra um rio largo e precisa atravessá-lo. Há uma solução engenhosa envolv endo a construção de uma balsa ou a utilização de objetos encontrados..
